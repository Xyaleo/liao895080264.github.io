#
前端是什么

前端开发是创建Web页面、app或小程序等前端界面的过程，利用HTML/ CSS/ DOM/ Flash/ JavaScript等各种Web技术以及衍生出来的各种框架，来实现互联网产品的用户界面交互。在过去，Web前端开发只是偏单调的网页制作，因为早期网站主要内容都是静态，以图片和文字为主，用户使用网站的行为也以浏览为主。随着互联网技术的发展，各种新的应用不断出现，前端开发的领域变得更加广泛。从手机，到智能手表，甚至人工智能，都需要用到前端技术。随着用户对界面的要求和功能的需求不断增加，前端技术也在不断的更新，以提高用户的交互体验。

#
前端的发展

最早于1990 年，Tim 以超文本语言 HTML 为基础在 NeXT 电脑上发明了最原始的 Web 浏览器。

在Web1.0时代，由于技术的限制，大部分网站只能呈现简单的图文信息，并不能满足用户在界面上的需求，对界面技术的要求也不高。随着硬件的完善、高性能浏览器的出现和宽带的普及，技术可以在用户体验方面实现更多种可能，前端技术领域迸发出旺盛的生命力。 　　2005年以后，互联网进入Web2.0时代，各种类似桌面软件的Web应用大量涌现，前端由此发生了翻天覆地的变化。网页不再只是承载单一的文字和图片，各种富媒体让网页的内容更加生动，网页上软件化的交互形式为用户提供了更好的使用体验，这些都是基于前端技术实现的。 　　如今，Web3.0仍是一个正在尝试概念。其中，网站内的信息可以直接和其他网站相关信息进行交互，能通过第三方信息平台同时对多家网站的信息进行整合使用；用户在互联网上拥有自己的数据，并能在不同网站上使用。Web3.0意味着前端仍需要继续发展，在未来Web前端开发依旧是一个十分重要的领域。

#
前端开发的职责

  1. 使用HTML+CSS并结合Javascript负责产品的前端开发和页面制作。
  2. 负责相关产品的需求以及前端程序的实现，提供合理的前端架构。改进和优化开发工具、开发流程、和开发框架。
  3. 与产品、后台开发人员保持良好沟通，能快速理解、消化各方需求，并落实为具体的开发工作 ；能独立完成功能页面的设计与代码编写，配合产品团队完成功能页面的需求调研和分析。
  4. 了解服务器端的相关工作，在交互体验、产品设计等方面有自己的见解。
  5. 跟踪了解最新的Web前端技术，做到与时俱进。

#
前端核心技术

# HTML

HTML是一种标记语言，能够实现Web页面并在浏览器中显示。HTML5作为HTML的最新版本，引入了多项新技术，大大增强了对于应用的支持能力，使得Web技术不再局限于呈现网页内容。

# CSS

CSS是一种用来表现HTML或XML等文件样式的计算机语言。CSS不仅可以静态地修饰网页，还可以配合各种脚本语言动态地对网页各元素进行格式化。

# JavaScript

JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML网页上使用，用来给HTML网页增加动态功能。
